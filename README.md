# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview

This project predicts car prices using Machine Learning. It demonstrates the complete ML workflow, including data preprocessing, data cleaning, feature engineering, model training, evaluation, and an interactive Streamlit dashboard.

## Features

- Data Cleaning
- Data Preprocessing
- Data Visualization
- Feature Engineering
- Linear Regression Model
- Model Evaluation
- Interactive Streamlit GUI
- Car Price Prediction

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn

## Dataset

UCI Machine Learning Repository - Automobile Dataset

## Installation

```bash
pip install -r requirements.txt
```

Run the training script:

```bash
python project.py
```

Run the Streamlit application:

```bash
streamlit run gui.py
```

## Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
